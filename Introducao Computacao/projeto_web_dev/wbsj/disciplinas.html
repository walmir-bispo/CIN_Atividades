DISCIPLINAS

Seção de exibição das disciplinas do curso de Ciência da Computação

O Curso de Ciência da Computação do CIn surgiu na década de 70, preparando os alunos para o mercado de trabalho e para a carreira acadêmica, sempre com foco em uma formação de excelência.

O profissional de Ciência da Computação é habilitado a desenvolver sistemas de software utilizando técnicas avançadas para resolver problemas reais, aplicando soluções que envolvam computação, independente de qual seja lá qual for o ambiente.
